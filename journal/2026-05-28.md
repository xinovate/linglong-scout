### 8. Redis 缓存 + 定时早报

**目标**：早报缓存从文件系统迁移到 Redis，新增 CLI 命令供 cron 定时触发。

**方案**：
- 新增 CLI 命令 `linglong-scout brief` — 独立生成早报，支持 `--force` 强制重新生成
- 缓存：`scout:brief:{date}` → Redis string, TTL 25h
- 去重历史：`scout:history:{date}` → Redis hash, 保留 14 天
- MCP `generate_brief` 优先查 Redis 缓存，无则生成
- `redis>=5.0` 从可选依赖升级为必选
- 服务器 cron：`55 6 * * * linglong-scout brief`
